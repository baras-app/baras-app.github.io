## Setting Up Overlays

## Enabling Overalys

Navigate to the **Overlays** tab in order to view overlays options.

By default all overlays are disabled. Click any overlay button to enable it. Once at least one overlay is enabled
you can toggle it's visiblity state by clicking **Hidden**. You should see the button change to visible.

## Moving Overlays

Click the **Locked** button in order to toggle **Move Mode**. Once in move mode all enabled overlays
will change their appearance to semi-transparent with a grey border.

Click anywayer on the overlay to drag it to the desired posistion. Click and drag the handle in the bottom right corner to resize.

:::note
Overlay posistions will not save until you exit move mode.
:::

## Customizing Overlays

Overlay's appearance and settings can be customzied by clicking the **Settings** button
at the top left of overlay page.

Overlay profiles can also be set here.

:::note
Changes will only persist if you click save
:::
