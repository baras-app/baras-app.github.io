# Features Overview

BARAS (Battle Analysis and Raid Assessment System) is a combat log parser and overlay for Star Wars: The Old Republic.

## Coming Soon

Detailed feature documentation is being written. Check back soon!

## Quick Preview

- **Real-time overlays** - See combat metrics during gameplay
- **DPS/HPS tracking** - Monitor damage and healing output
- **Raid analysis** - Detailed breakdown of group performance
- **Timers** - Track boss abilities and cooldowns
